# 03_7 Validación: Caso Finanzas Globales (Fallo/Mejora)

A diferencia del clima, el mercado financiero es mucho más difícil de predecir.

## Resultados del Test:
*   **Estado:** Fallido en el criterio **C4 (Validez)**.
*   **Problema:** El modelo intentó usar un "calendario" para la bolsa (como si las crisis financieras ocurrieran cada diciembre). La realidad es que las finanzas no son cíclicas como las estaciones.
*   **Aprendizaje:** Para validar la H1 en finanzas, necesitamos un "Cerebro Macro" más complejo que no dependa solo del tiempo, sino de otros indicadores económicos.
