# 01_6 Tablas de Correspondencia de Medicion (Con Regla)

## 1. Propiedad -> Indicador -> Metrica -> Regla
- Emergencia -> persistencia macro -> estabilidad de atractores -> perdida de prediccion sin nivel macro.
- Cohesion -> dependencia interna vs externa -> informacion mutua -> diferencia positiva significativa.
- No-localidad -> causalidad distribuida -> centralidad distribuida -> sin fuente dominante.
- Persistencia -> continuidad temporal -> cambios de regimen -> patrones estables en ventanas largas.
- Causalidad descendente -> restriccion macro -> contrafactuales -> diferencias micro significativas.

## 2. Nivel -> Datos -> Tecnica
- Micro -> datos individuales -> SBA + distribuciones.
- Meso -> redes -> metricas de grafos.
- Macro -> series globales -> modelos dinamicos.
