# Protocolo de Moderación y Resolución

En esta dinámica no existen jueces externos. El Auditor y el Defensor deben autoregularse siguiendo este protocolo.

**Criterios de Calidad (La "Constitución")**
En caso de desacuerdo, prevalece la opción que mejor cumpla estos criterios, en este orden:

1.  **Claridad:** Si un párrafo es ambiguo para un lector culto pero no experto, debe reescribirse. La oscuridad no es profundidad.
2.  **Precisión:** Los términos técnicos (EDI, Sinergética, Hiperobjeto) deben usarse con rigurosidad matemática/filosófica.
3.  **Concisión:** Si se puede decir con menos palabras sin perder significado, se corta. (Navaja de Ockham estilística).
4.  **Coherencia:** El tono debe ser consistente (Académico, objetivo, firme pero humilde).

**Resolución de Bloqueos**
Si tras 3 interacciones (Crítica -> Respuesta -> Réplica -> Contraréplica) no hay acuerdo:
1.  Se marcan ambas versiones como "DISPUTA" en el archivo de cierre.
2.  Se solicita (simulado) la intervención del "Director de Tesis" (usuario/humano) para decidir.

**Tono de la Interacción**
- **Profesional y Constructivo:** El objetivo es mejorar el texto, no ganar un debate.
- **Sin Egos:** El texto no es del Defensor, es de la Tesis.
- **Auditor:** No solo destruye, propone alternativas.
- **Defensor:** No se atrinchera, busca la mejor versión posible.
