# Datos y Fuentes

Objetivo: completar el calculo de cohesion (CR) mediante proxies de interaccion externa.

Fuentes recomendadas:
- OWID COVID-19 data (casos/defunciones).
- Google Community Mobility Reports (movilidad agregada).
- World Bank: Air transport passengers (IS.AIR.PSGR).

Estas fuentes permiten construir una medida de acoplamiento externo y completar el indicador de symploke.
