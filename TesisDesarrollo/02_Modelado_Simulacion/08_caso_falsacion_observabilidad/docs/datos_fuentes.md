# Datos y Fuentes

Objetivo: sustituir la observabilidad insuficiente con series continuas y proxies robustos.

Fuentes recomendadas:
- GDELT 2.1 (eventos globales y cobertura diaria).
- Wikipedia Pageviews (actividad agregada).
- World Bank WDI (indicadores estructurales con historico amplio).

Estas fuentes pueden elevar la cobertura y habilitar el calculo de EDI y CR.
