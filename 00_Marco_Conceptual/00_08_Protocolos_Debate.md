# 00_08 Protocolos de Debate: API de Comunicación

Cuando discutimos la tesis con otros investigadores, usamos este protocolo de comunicación para evitar "Race Conditions" conceptuales.

1.  **Handshake:** Definir términos antes de empezar (Usar Glosario 00_02).
2.  **Request:** Toda objeción debe venir acompañada de una métrica alternativa.
3.  **Response:** La respuesta debe incluir una prueba operativa (C1-C5).
4.  **Error Handling:** Si no hay acuerdo en los datos, se vuelve a la Fase Sintética.
