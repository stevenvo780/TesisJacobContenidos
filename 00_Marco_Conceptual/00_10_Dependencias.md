# 00_10 Dependencias: El package.json Conceptual

Para que este proyecto compile, necesita las siguientes "librerías" teóricas:

```json
{
  "dependencies": {
    "Realismo_Estructural": "^2.0",
    "Teoria_de_Sistemas": "latest",
    "Termodinamica_No_Lineal": "v3.1",
    "Estadistica_Bayesiana": ">=1.5"
  },
  "devDependencies": {
    "Filosofia_Analitica": "para el debugging",
    "Python_Numpy": "para la ejecución"
  }
}
```

Si alguna de estas dependencias externas se actualiza o se refuta, debemos parchear nuestra tesis.
